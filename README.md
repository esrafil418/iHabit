# Habit Tracker

A modern habit tracking application built with React and TypeScript. Track your daily habits, maintain streaks, and visualize your progress over time.

## ✨ Features

- **Daily Habit Tracking**: Mark habits as complete for any day of the week
- **Streak Tracking**: Automatically calculates and displays your current streak for each habit
- **Weekly View**: Navigate through weeks to track progress over time
- **Local Storage**: All data is saved locally in your browser
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## 🚀 Technologies

- React 18+
- TypeScript
- Tailwind CSS
- date-fns for date manipulation
- Local Storage for data persistence
